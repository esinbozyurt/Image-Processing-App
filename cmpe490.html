Image Processing User Manual

Welcome!

This user manual will guide you through the Image Processing application created using the Matlab Design App. Before you begin, please review the information below to understand the basic features of the application.

Open Image Button

Use the "Open Image" button to start the application. This button allows you to upload an image with the PNG extension. If your image does not have the PNG extension, you will immediately receive an error message on the screen. Please check your image format before proceeding.


Select Button

The "Select" button in the upper left corner offers various filter options. Once you have determined the filter you want to use, click the "Select" button to apply it.


Process Button

After selecting the filter, adjust the parameters (such as Noise Mean and Noise Variance) using the sliders provided. Once you are satisfied with the settings, click the "Process" button to apply the filter to your image.


Save Image Button

Use the "Save Image" button to save the processed image to your device. You can specify the desired location for saving the image.


Help Button

If you encounter any problems or need additional assistance while using the application, click the "Help" button for a detailed user guide.



Usage Stages

Image Upload

Click the "Open Image" button to upload a PNG image. If you upload an image in an incompatible format, an error message will appear.

Filter Selection

Choose the desired filter from the options available using the "Select" button in the upper left corner.

Starting Process

After selecting the filter, adjust the filter parameters using the sliders provided. Once you are satisfied with the settings, click the "Process" button to apply the filter to your image.

Saving Image

Use the "Save Image" button to save the processed image to your device. Specify the desired location for saving the image.

Getting Help

If you encounter any problems or need additional assistance, click the "Help" button for a detailed user guide.

By following these simple steps, you can easily use the Image Processing application. For more detailed information, please refer to the comprehensive user guide.

Enjoy your work!
